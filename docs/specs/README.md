# Feature Specifications

Подробные спецификации для будущих фич платформы онлайн-курсов.

## Статус реализации

### ✅ Реализовано
- Аутентификация и авторизация (JWT, роли)
- Управление пользователями (CRUD, роли)
- Управление курсами (CRUD, модули, уроки, шаги)
- Управление видео (загрузка, хранение MinIO)
- Админ-панель (пользователи, курсы, видео)
- Пагинация и фильтры для всех списков

## Структура спецификации

Каждая спецификация содержит:

1. **Описание** - что делает фича
2. **Цели** - зачем нужна
3. **Требования** - функциональные и нефункциональные
4. **Архитектура** - структура БД, API, сервисы
5. **План реализации** - пошаговый план
6. **API спецификация** - эндпоинты и контракты
7. **UI/UX** - описание интерфейса
8. **Тестирование** - сценарии тестирования
9. **Зависимости** - от каких фич зависит
10. **Оценка** - время реализации

## Как использовать

1. Выбери фичу из списка
2. Прочитай спецификацию
3. Следуй пошаговому плану реализации
4. Обновляй статус в этом файле

## Соглашения

- Все спецификации на русском языке
- Код и API на английском
- Используем существующие паттерны проекта
- Следуем принципам Clean Architecture
- Пишем тесты для каждой фичи
