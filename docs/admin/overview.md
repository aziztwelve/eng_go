# Admin Panel Overview

## Описание

Административная панель предоставляет полный контроль над платформой изучения английского языка. Позволяет управлять пользователями, курсами, видео контентом и мониторить активность.

## Основные возможности

### 1. Dashboard
- Статистика по пользователям
- Статистика по курсам
- Статистика по видео
- Последние активности

### 2. User Management
- Просмотр списка пользователей
- Редактирование ролей (admin, instructor, student)
- Удаление пользователей
- Поиск и фильтрация

### 3. Course Management
- Создание и редактирование курсов
- Управление модулями
- Управление уроками
- Управление шагами (text, video, quiz)
- Публикация/снятие с публикации
- CEFR уровни (A1-C2)

### 4. Video Management
- Загрузка видео файлов
- Просмотр списка видео
- Редактирование метаданных
- Удаление видео
- Интеграция с MinIO

## Роли и права доступа

### Admin
- Полный доступ ко всем функциям
- Управление пользователями
- Управление курсами
- Управление видео

### Instructor
- Создание и редактирование своих курсов
- Загрузка видео
- Просмотр статистики своих курсов

### Student
- Нет доступа к админ панели
- Только пользовательский интерфейс

## Технические детали

### Backend Architecture

```
┌─────────────┐
│   Gateway   │ ← HTTP/REST API
└──────┬──────┘
       │
       ├─────────────┐
       │             │
   ┌───▼────┐   ┌───▼────┐
   │  Auth  │   │  User  │
   │Service │   │Service │
   └────────┘   └────────┘
       │             │
   ┌───▼────┐   ┌───▼────┐
   │ Course │   │ Video  │
   │Service │   │Service │
   └────────┘   └────────┘
       │             │
   ┌───▼─────────────▼───┐
   │    PostgreSQL       │
   └─────────────────────┘
           │
       ┌───▼────┐
       │ MinIO  │
       └────────┘
```

### Database Schema

- **users** - пользователи (auth-service)
- **profiles** - профили (user-service)
- **courses** - курсы
- **modules** - модули курсов
- **lessons** - уроки
- **steps** - шаги уроков
- **videos** - метаданные видео

### API Endpoints

Все admin endpoints защищены middleware:
- Проверка JWT токена
- Проверка роли (admin/instructor)

Базовый путь: `/api/v1/admin`

## Мониторинг

Логи доступны в `./logs/`:
- `auth.log` - Auth Service
- `user.log` - User Service
- `course.log` - Course Service
- `video.log` - Video Service
- `gateway.log` - API Gateway

## Troubleshooting

### Проблемы с доступом
1. Проверьте JWT токен
2. Убедитесь что роль пользователя = admin
3. Проверьте логи Gateway

### Проблемы с загрузкой видео
1. Проверьте MinIO доступность
2. Проверьте размер файла (лимит 100MB)
3. Проверьте логи video-service

### Проблемы с базой данных
1. Проверьте подключение к PostgreSQL
2. Проверьте права доступа пользователей БД
3. Проверьте применены ли миграции
